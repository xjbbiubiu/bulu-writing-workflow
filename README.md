# bulu-writing-workflow

用于“布鲁成长手记”的每日夜报/热点复盘分阶段写作工作流。

这个 skill 把公众号文章写作拆成 `new`、`input`、`specify`、`outline`、`apply`、`review`、`final`、`cover` 八个阶段，核心目标是先收集素材、再定主线、再写正文，避免一上来就写成新闻拼盘。

## 文件

- `SKILL.md`: skill 主说明和工作流规则
- `scripts/new.py`: 创建当天草稿空间的脚本
- `agents/openai.yaml`: agent 配置

## 默认草稿目录

```text
/Users/Admin/Documents/bulu
```

使用时触发：

```text
bulu-writing-workflow
```
